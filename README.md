## 🚀 快速开始
1. 克隆仓库  
   ```bash
   git clone https://github.com/yijiu2333/Python_attendance-display.git
   cd Python_attendance-display
   ```

2. 安装依赖
   ```bash
   python -m venv venv
   venv\Scripts\activate
   pip install -r requirements.txt
   ```

3. 本地启动
   ```bash
   python mainpage.py

   ```

4. 打开浏览器访问
    http://127.0.0.1:8050/
